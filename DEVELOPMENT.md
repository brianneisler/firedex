# Development

# Requirements

* node.js 8.0+
* npm 5.0+

*NOTE:* we recommend using nvm
```sh
brew install nvm
nvm install 8
nvm use 8
```

## Setup

```sh
git clone https://github.com/brianneisler/firedex.git
cd firedex
npm install
```


## Build

To build firedex

```sh
npm run build
```


## Clean

To clean firedex

```sh
npm run clean
```


## Watch

To watch firedex and build on every file change

```sh
npm run watch
```

## Test

To run tests against firedex

```sh
npm test
```
